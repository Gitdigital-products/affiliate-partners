## üöÄ How to Add a New Partner
You don't need to edit the HTML to add a new partner. Simply update the data file:

1. Open `partners.json`.
2. Add a new object to the array following this format:
   ```json
   {
       "name": "Partner Name",
       "category": "SaaS / Marketing / Development",
       "commission": "X%",
       "link": "[https://your-affiliate-link.com](https://your-affiliate-link.com)"
   }
   ```
