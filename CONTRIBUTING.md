# Contributing to GitDigital Affiliate Partners

First off, thank you for considering contributing! 

## Ways to Contribute
* **Add Partners:** Submit a Pull Request updating `partners.json`.
* **Fix Bugs:** Check the "Issues" tab for broken links or UI bugs.
* **Funding:** If you can't code, consider sponsoring us via the "Sponsor" button!

## Pull Request Process
1. Fork the repo and create your branch.
2. If adding a partner, ensure the link is valid and the category is correct.
3. Submit the PR, and we will review it within 48 hours.

By contributing, you agree that your contributions will be licensed under the project's MIT License.
